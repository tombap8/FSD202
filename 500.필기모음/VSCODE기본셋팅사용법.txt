[ vscode 사용 단축키+방법 ]

1.  테마(화면색)변경
 - F1을 눌러서 검색나오면 "theme"이라고 친다
 - 색테마(Color theme)를 클릭
 - 여러 제시옵션에서 선택한다

2. 글자크기(확대/축소) : 
   ctrl+플러스버튼/마이너스버튼

3. 코딩시 소스선택 (html, css)
  1) 소스 라인 이동: alt + 위아래방향키
  2) 소스 라인 복제: alt + shift + 위아래방향키
  3) 라인선택 : ctrl+L 
(라인선택후 shift키를 누르고 위아래방향키-여러줄선택)
  4) 커서를 여러라인에 따로 두기 : 
	alt키를 누르고 소스를 클릭
	(연속라인은 alt+shift)
  5) 다음항목추가 : ctrl+D
	- 같은 문자를 찾아이동 선택
	- 태그 앞에서는 뒷태그를 같이 선택

4. 주석
- 전체주석 : ctrl + /
- 부분주석 : alt + shift + A
 ( 아쉬워.... 에이...)

5. 사이드 패널 열기/닫기 : ctrl + B


[ vscode 의 단축키 지정하기 ]
- ctrl+K S
-> K는 Kit 기능모음 중 S는 Shorcut 단축키 열기
-> 모음열기가 열리면 여기서 원하는 기능을 검색한다.

1. "모두 저장" 검색후 단축키 지정
->플러스 버튼을 클릭하여 원하는 키조합을 누르고
사용하는 곳이 없는 키라면 이때 Enter를 쳐서 지정함!
(지정키는 ctrl+alt+S)

-> 지정된 곳에서 마우스 오른쪽버튼을 클릭하면
	변경, 삭제등  할 수 있음!
