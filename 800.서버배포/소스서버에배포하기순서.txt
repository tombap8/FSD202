[ 소스 서버 배포하기 ]

1. 필수확인사항: 
  - 호스팅 서버의 접속 정보를 확인 
   (닷홈에 로드인 후 "마이닷홈"에서 "웹호스팅목록"
   에서 자기가 올릴 서버의 접속정보를 확인한다.)
  - 소스파일 올리기 위한 FTP 정보:

   예)  "웹서버/FTP 정보"항목에서
   웹서버 아이피	112.175.184.75
   FTP 아이디	tombap8
   FTP 암호 -> 만약 잊어버렸으면 "변경"함

  - DB정보:
  예) DB 정보 : DB 명 - tombap8 (아이디명과 동일)
	       DB 아이디 - tombap8 (아이디명과 동일)
	       DB 암호 -> 만약 잊어버렸으면 "변경"함

2. 웹소스 올리기
  1) FTP 프로그램을 열고 접속정보를 입력하여 접속함
  (알드라이브, 파일질러 등)
  2) 접속시 Root폴더를 잘 설정해야함
    - 닷홈은 Root폴더가 html 
       -> FTP설정시 서버기본경로에 /html/
    - 카페24는 Root폴더가 www
  3) 실제 배포할 파일을 업로드한다.
   - 웹경로를 잘 보고 URL주소를 잘 매칭하여 열어야함!
  (주의: 경로는 반드시 대소문자를 모두 구분함!-닷홈설정)

	예) 2가지 프로젝트 업로드

	(1) 파일럿 PJ : 올린후 폴더명을 "Pilot_PJ"로 변경
	D:\[탐쌤코딩스테이지_풀스택2회차]_원본\
	800.GitHub\FSD202\016.파일럿PJ\002.Source

	(2) 마이아이템 PJ
	C:\xampp\htdocs\mysite\MyItem_PJ

  -> DB가 사용되지 않는 파일은 업로드 후 바로 확인해 본다!
  -> 기본확인 URL은 http://아이디.dothome.co.kr/
  -> 폴더명 까지만 쓰면 하위 index파일을 찾아서 읽어옴
  예) http://tombap8.dothome.co.kr/Pilot_PJ
       http://tombap8.dothome.co.kr/MyItem_PJ/

3. DB 생성파일 만들기 (MyItem PJ 에서 사용)
 1) DB는 주어진 아이디명 DB를 사용하게됨(우리가 못만든다!)
 2) 테이블은 생성 및 데이터 입력 SQL 쿼리문을 
	로컬 개발 셋팅에서 추출한다!
 3) MYSQL 어드민 모드 페이지에서 해당DB(mydb)의
   두가지 테이블을 추출한다.
  - member 테이블, board_free 테이블
  - 각 테이블을 선택하여 탭메뉴에서 "내보내기"메뉴를 선택함
  - 여기서 "실행" 버튼을 클릭하면 다운로드 폴더에 자동생성됨
  - 내용은 테이블 생성 및 데이터 넣기 쿼리문으로 구성됨!
  - 별도의 폴더에 잘 저장한다!

4. 호스팅 DB에 접속하기
(1) 호스팅업체 홈페이지에서 접속정보를 확인한다.
예) url은? 
    제공 내역 에서
  MySQL 관리(UTF-8) : tombap8.dothome.co.kr/myadmin
  (접속장애시 아래↓ 주소로 접속하세요)
  MySQL 관리(EUC-KR) : tombap8.dothome.co.kr/myadmin2

(2) 관리화면에서 로그인한다.
(3) 내 아이디명으로 된 DB를 선택한다.
(4) SQL탭 메뉴를 선택후 아까 생성한 SQL파일의 내용을 복사 후 붙여놓고 실행한다!

5. 소스에서 DB접속 정보 중 아이디, 비번을 수정한다!
(1) C:\xampp\htdocs\mysite\MyItem_PJ\process\dbcon.inc
(2) C:\xampp\htdocs\mysite\MyItem_PJ\admin\DBconn.inc
(3) C:\xampp\htdocs\mysite\MyItem_PJ\board\DBconn.inc
