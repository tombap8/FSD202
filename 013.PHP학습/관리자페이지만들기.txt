[ 관리자모드 ]
1. 대상: 권한컬럼(auth) 의 값이 "A"인 사용자
2. 관리자 메뉴에 접근가능
 1) 페이지 상단 끝에 "관리자" 링크가 노출됨
 2) 링크 클릭시 사용자 리스트가 보이는 관리자 페이지 조회
 - 관리자 페이지에서도 관리자 auth값이 "A"임을 확인한다!
3. 관리자 페이지 생성하기
 1) Drama_PJ를 기본 모델로 사용한다
 2) Drama_PJ를 카피하여 MyItem_PJ폴더 하위에
    복사한다. 그리고 폴더명을 "admin"으로 변경한다.
 3) Drama_PJ의 기본 테이블을 "member"로 변경후
    리스트 페이지의 모든 항목을 "member"의 컬럼
    에 맞도록 매칭하여 변경한다.
 4) 모든 html의 컬럼명도 같이 변경해 준다.
